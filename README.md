# Echo - Overwatch Discord Bot

> **Status:** Work in Progress (WIP) - This project is under active development. New features are being added and the codebase may undergo changes and refactoring.

## Description

Echo is an unofficial Discord bot focused on utility features for Overwatch players. Built with Node.js, the project aims to bring information lookup, official data parsing, and interactive tools directly to Discord communities.

## Features

- **Patch Notes System:** Scrapes the official Blizzard website, organizes update notes into categories, and displays them through interactive menus (StringSelectMenu), avoiding Discord's message limitations while keeping the chat clean.

## Roadmap

- [ ] Integration with the OverFast API to retrieve player profiles, stats, and ranks.
- [ ] Commands for random hero compositions and hero suggestions.

## Running Locally

### Requirements

- [Node.js](https://nodejs.org/) (v16.14 or higher)
- A valid bot token from the [Discord Developer Portal](https://discord.com/developers/applications)
